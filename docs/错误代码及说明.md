## 常见错误排查

### 错误代码 7：NET_DVR_NETWORK_FAIL_CONNECT

**可能原因：**
1. 设备不在线或已关机
2. 网络连接问题（网线松动、交换机故障等）
3. IP 地址或端口号错误
4. 设备连接数已满（达到最大 128 个在线用户）
5. 之前的连接未正确释放
6. 防火墙阻止连接
7. 设备正在忙于其他操作

**解决方案：**
1. 检查设备是否在线（ping 设备 IP）
2. 确认 IP 地址和端口号正确（默认端口 8000）
3. 等待 30-60 秒后重试（让设备自动释放旧连接）
4. 关闭其他连接（Web 界面、客户端软件）
5. 检查防火墙设置
6. 重启设备（作为最后手段）

### 错误代码 5：NET_DVR_OVER_MAXLINK

**说明：** 连接到设备的用户个数超过最大（128 个）

**解决方案：**
1. 等待其他用户断开连接
2. 在设备 Web 界面强制断开部分连接
3. 重启设备清除所有连接

### 错误代码 1：NET_DVR_PASSWORD_ERROR

**说明：** 用户名或密码错误

**解决方案：**
1. 检查用户名和密码是否正确
2. 注意密码大小写
3. 检查是否需要修改默认密码
