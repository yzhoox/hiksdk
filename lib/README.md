# 海康威视 SDK 动态链接库

本目录包含海康威视官方 SDK 的动态链接库文件。

## 📁 目录说明

- **Windows/** - Windows 平台库文件（DLL）
- **Linux/** - Linux 平台库文件（SO）

## ⚠️ 重要

**所有库文件已被 Git 托管，请勿删除或修改！**

这些库文件是 SDK 运行的必要依赖，CGO 编译时会链接这些库。

## 📦 版本信息

- SDK 版本：海康威视网络 SDK V6.x
- 支持平台：Windows x64, Linux x64
